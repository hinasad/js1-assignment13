Q:1 What is JavaScript ?

Ans: JavaScript is a scripting language that enables you to create dynamically updating content,
control multimedia, animate images, and pretty much ...

Q:2 Write down difference between Declaration and Initialization a variable ?

Ans: Declarations specify the type followed by the name, and optionally the initialization. Initialization sets the variable to a new instance.

1-Declaration

var a;

2-Initialization

var a = 8;

Q:3 In how many ways we can define a variable?

Ans: Basically we can declare variables in three different ways by using var, let and const keyword.

Q:4 Is variable is case sensitive ?

Ans: JavaScript is a case-sensitive language.

Q:5 When we declare a variable can we start it with any number?

Ans: JavaScript variable names should not start with a numeral.
